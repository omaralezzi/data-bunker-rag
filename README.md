# 🧠 Data Bunker RAG
### Offline AI Knowledge & Survival Intelligence System

---

## 🚀 Overview

**Data Bunker RAG** is a standalone, offline-first Retrieval-Augmented Generation (RAG) system designed to deliver intelligent, context-aware answers without relying on internet connectivity.

This project is part of a larger system architecture (**Data Bunker**) aimed at building resilient, self-contained knowledge platforms for emergency, field, and offline environments.

It combines local AI models, vector search, and curated knowledge sources into a unified system capable of reasoning over stored information.

---

## ⚙️ Core Capabilities

- 🔍 **Semantic Search (Vector-based retrieval via Qdrant)**
- 🧠 **Local AI reasoning (Ollama LLM integration)**
- 📚 **Offline knowledge access (Kiwix-based sources)**
- 🌍 **Multi-domain intelligence system**
  - Water purification
  - Medical emergencies
  - Power generation
  - Food preservation
  - Survival / preparedness
- 🌐 **Multilingual support**
  - Arabic 🇸🇦
  - English 🇬🇧
  - Auto-detection mode
